# Privacy Policy - TikTok Viral Clothing Bot

**Last Updated: March 8, 2026**

## 1. Introduction

This Privacy Policy describes how TikTok Viral Clothing Bot ("we", "our", "the Service") collects, uses, and protects information when you use our service.

## 2. Information We Collect

### 2.1 Public TikTok Data
We collect only publicly available information from TikTok:
- Video URLs and IDs
- Video thumbnails (publicly accessible images)
- Public engagement metrics (views, likes, comment count)
- Public comments text
- Video post timestamps
- Public hashtags
- Public creator usernames (no private details)

### 2.2 What We DON'T Collect
We do NOT collect:
- ❌ Personal identifiable information (PII)
- ❌ Email addresses
- ❌ Phone numbers
- ❌ IP addresses
- ❌ Location data
- ❌ Private videos or content
- ❌ User passwords or credentials
- ❌ Private messages
- ❌ Device information
- ❌ Cookies or tracking data

## 3. How We Collect Data

Data is collected through:
- TikTok Official API (when available)
- Publicly accessible TikTok content only
- Automated searches for fashion-related hashtags
- Public comment reading (no authentication required)

## 4. How We Use Data

Collected data is used exclusively for:
- **Fashion Trend Analysis**: Identifying viral clothing content
- **Business Insights**: Understanding customer product inquiries
- **Analytics**: Measuring engagement on fashion videos
- **Dashboard Display**: Presenting trends in web interface

We do NOT:
- ❌ Sell data to third parties
- ❌ Share data with advertisers
- ❌ Use data for marketing purposes
- ❌ Create user profiles
- ❌ Track individual users

## 5. Data Storage

### 5.1 Storage Method
- Data is stored locally in SQLite database
- Database location: Local device only
- No cloud storage or external servers

### 5.2 Data Retention
- Videos older than 24 hours are automatically deleted
- Database can be manually cleared at any time
- No long-term data retention

### 5.3 Data Security
- Local storage only (not transmitted)
- No external API exposure
- Database file protected by file system permissions
- No encryption needed (public data only)

## 6. Data Sharing

We do NOT share data with:
- Third-party services
- Advertisers
- Data brokers
- Analytics companies
- Other users

**Exception**: The web dashboard displays aggregated public data locally (localhost only, not accessible externally).

- **Opt-Out**: Stop using the Service at any time
